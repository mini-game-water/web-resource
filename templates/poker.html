게임 규칙

텍사스 홀덤 기본 규칙

  • 각 플레이어에게 2장의 홀 카드가 비공개로 배분됩니다.
  • 테이블 중앙에 5장의 커뮤니티 카드가 단계별로 공개됩니다.
  • 홀 카드 2장과 커뮤니티 카드 5장 중 최강의 5장 조합을 만드세요.

게임 진행 순서

  1. 프리플롭 — 홀 카드 2장 배분 후 첫 베팅
  2. 플롭 — 커뮤니티 카드 3장 공개 후 베팅
  3. 턴 — 4번째 커뮤니티 카드 공개 후 베팅
  4. 리버 — 5번째 커뮤니티 카드 공개 후 최종 베팅
  5. 쇼다운 — 남은 플레이어의 패를 비교하여 승자 결정

베팅 액션

  • 폴드 — 카드를 버리고 해당 라운드 포기
  • 체크 — 추가 베팅 없이 넘기기 (앞 베팅이 없을 때)
  • 콜 — 앞 사람의 베팅 금액과 동일하게 맞추기
  • 레이즈 — 베팅 금액을 높이기
  • 올인 — 보유 칩 전부를 베팅

기본 설정

  • 시작 칩: 10,000 코인
  • 빅 블라인드: 1,000 코인 / 스몰 블라인드: 500 코인
  • 딜러(AI)가 항상 참여하여 최소 1:1 대결이 보장됩니다.
